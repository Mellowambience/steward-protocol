# Review

This folder holds the consistency machinery for The Steward Project.

- [`CONSTITUTIONAL_REVIEW.md`](CONSTITUTIONAL_REVIEW.md) — the rubric run on every
  proposed change and every implementation. Answers: *"Does this still reflect
  the constitution?"*

The reviewer's role (human, agent, or both) is to apply this check consistently
as the project grows. It is more valuable than writing more spec.
