# Reference — Implementations

A directory of systems that implement (or aim to implement) the Steward Protocol.

- **AetherTwin** — the reference implementation (flagship).
- **Fairy OS** — the operating-system layer (consent, local-first, no hidden net).
- **MIST** — a local sovereign companion; see `examples/self-assessment.md`.

To list your implementation, add a short entry and a self-assessment link.
Interoperability over lock-in: any system speaking this language belongs here.
