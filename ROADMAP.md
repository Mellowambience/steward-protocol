# Roadmap

The order is deliberate. We build the soil before the trees.

- **Phase 0 — Write the Constitution.** *(in progress)* The words that outlast us.
- **Phase 1 — Steward Protocol v1.** This repository: README, Constitution,
  Principles, Language, Standard, Manifesto, FAQ, License.
- **Phase 2 — The Language.** Disseminate the shared vocabulary; adopt it in our own work.
- **Phase 3 — Steward Compatible.** Badge program + public self-assessments.
- **Phase 4 — Reference Architecture.** AetherTwin implements the protocol; Fairy OS,
  Genesis, and Agent Studio follow.
- **Phase 5 — Living Atlas.** Explorable knowledge map; ideas as places, projects as stars.
- **Phase 6 — Stewardship Benchmark.** Measure stewardship, not IQ.
- **Phase 7 — The Guild.** A living workshop of builders; apprentices become peers.
- **Phase 8 — Steward Foundation.** A nonprofit: open governance, community voting,
  independent of any company.

> For the next stretch, resist the urge to build another product. Build the soil
> from which hundreds of products could grow.
