# The Steward Project

> An open, evolving protocol for human–AI collaboration.

Not a company first. Not a model. Not another AI app. **A protocol.**

The internet has TCP/IP. Open source has its definition. The Steward Project
defines what *responsible collaborative intelligence* looks like — so that
people, AI systems, and organizations can work together in a shared language
regardless of which model, operating system, or company they use.

## The four layers

1. **Constitution** — The Steward Protocol. Principles, ethics, design philosophy. *Stable. Enduring. Foundational.*
2. **Language** — A shared vocabulary ([`LANGUAGE.md`](LANGUAGE.md)). *A common language for humans and intelligent systems.*
3. **Standard** — Open implementation specs ([`STANDARD.md`](STANDARD.md)). *Build it your way. Stay aligned on what matters.*
4. **Ecosystem** — Interoperable and open. *Many implementations. One shared foundation.*

## Status

We are at **Phase 0–1**: the soil. The words come before the software.
[AetherTwin](https://mellowambience.github.io) is the *reference implementation*
— one beautiful example, not the project itself. Other builders are welcome to
implement the protocol in coding assistants, tutors, scientific collaborators,
creative companions, and robotics.
